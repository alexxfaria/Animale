# Animale Hospital Veterinário 24h

Landing page estática, responsiva e compatível com GitHub Pages.

## Estrutura

- `index.html`: estrutura da landing page
- `styles.css`: estilos, responsividade e animações
- `script.js`: menu mobile e animações de entrada
- `assets/logo.png`: logo atual da clínica

## Publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie os arquivos deste projeto para a branch principal, normalmente `main`.
3. No GitHub, abra `Settings`.
4. Acesse `Pages`.
5. Em `Build and deployment`, selecione `Deploy from a branch`.
6. Escolha a branch `main` e a pasta `/ (root)`.
7. Salve e aguarde a URL pública ser gerada.

## Ajustes recomendados

- Atualize `og:url` em `index.html` com a URL real do repositório publicado.
- Se quiser uma imagem específica para compartilhamento, troque `og:image`.
- Revise links de mapa e endereço se quiser apontar para uma localização exata.

## Observação

Os links usados na seção de Instagram apontam para posts públicos encontrados do perfil para servir como exemplos reais de conteúdo.
